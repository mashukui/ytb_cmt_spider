👋 软件简介 👋 Overview

ytb_cmt_spider 是一款面向 YouTube 评论数据采集场景的桌面 GUI 工具。 支持批量输入视频链接，自动采集视频评论、评论作者、评论时间、点赞数、语言识别结果等数据， 并导出为 CSV / JSON 文件。用户无需配置 Python 环境，下载客户端后登录即可使用。

ytb_cmt_spider is a desktop GUI tool for YouTube comment data collection. Batch input video URLs and automatically crawl comments, author info, timestamps, like counts, and language detection results — all exported as CSV or JSON. No Python setup required.

❓ 常见问题 ❓ FAQ

🟢 是否需要安装 Python？
不需要。软件已打包为桌面客户端，下载对应系统版本后即可运行。
🟢 Do I need to install Python?
No. The software is packaged as a standalone desktop app — just download and run.
🟢 是否需要登录 YouTube？
不需要。软件通过网页接口实现持续采集，无需登录 YouTube 账号。
🟢 Do I need a YouTube account?
No. The tool collects data via public web interfaces — no YouTube login required.
🟢 采集中断后数据会丢失吗？
不会。软件在采集过程中持续保存结果，网络异常或任务中断不会丢失已采集的数据。
🟢 Will I lose data if crawling is interrupted?
No. Results are saved continuously — network issues or interruptions won't wipe collected data.
🟢 支持导出哪些格式？
支持 CSV 和 JSON 两种格式。CSV 适合 Excel / WPS / Numbers 直接打开，JSON 适合程序化处理和接口对接。
🟢 What export formats are supported?
CSV and JSON. CSV opens directly in Excel / WPS / Numbers. JSON is ideal for programmatic use and API integration.
