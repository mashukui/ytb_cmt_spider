常见问题

以下问题会直接影响采集结果，建议购买前阅读。

是否需要安装 Python？

不需要。软件已打包为 Windows 和 macOS 桌面客户端，下载对应版本后解压运行。

是否需要登录 YouTube？

不需要登录 YouTube 账号。软件采集公开可见的评论，但本机网络仍需能够正常访问 YouTube。

为什么实际结果少于评论区显示数量？

评论区关闭、部分评论不可见、视频访问受限、页面接口变化、网络中断或平台规则都可能影响结果。-1 表示尝试采集全部可见评论，不保证得到页面显示的完整数量。

中断后已采集数据会丢失吗？

软件会在采集过程中持续保存结果。网络异常或任务中断不会把已经写入文件的内容整体清空。采集时不要打开结果文件，以免文件占用影响继续写入。

支持导出哪些格式？

支持 CSV 和 JSON。CSV 适合 Excel、WPS、Numbers 和 pandas，JSON 适合程序处理、数据库导入或接口对接。

GitHub 仓库是否包含源代码？

不包含。该仓库用于产品说明、版本发布、客户端下载和问题反馈，软件以商业授权方式提供。
